数据类型（支持6种）

  • 数字（例如 1.2、13、10px）
  • 文本字符串，无论是否有引号（例如 "foo"、'bar'、baz）
  • 颜色（例如 blue、#04a3f9、rgba(255, 0, 0, 0.5)）
  • 布尔值（例如 true、false）
  • 空值（例如 null）
  • 值列表，用空格或逗号分隔（例如 1.5em 1em 0 2em、Helvetica, Arial, sans-serif）
