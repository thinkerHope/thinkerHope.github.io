题外

JS escaped & unescaped output

对XSS attack很有用。

一个栗子

有一个页面允许用户输入地址，并且在另外一夜确定，如果用户输入

下个页面将会简单地运行这个脚本，特别是当攻击者把脚本放在一个无限循环里，程序就崩了。

因此，用户输入的文本将会被解码(escaped)，对应的就是编码(uneasped)。

这样，浏览器就会把他当做一个HTML元素而不是script元素了。
